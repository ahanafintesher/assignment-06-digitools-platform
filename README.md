# Digitools

A modern digital marketplace for premium tools, templates, and AI-powered services. Designed to help users boost productivity and streamline their workflow in one place.

## Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- DaisyUI
- React-Toastify (NPM Package)
- JSON (for product data)

## Features

- Curated collection of digital products with pricing and categories
- Responsive and clean UI for smooth user experience
- Product cards with tags, features, and purchase options
